Encodes text by substituting character with another one provided in the pair.(pass
    the substitutions array like ['ab','cd'] and input string)and return the encoded string.
    For example pair "ab" defines all "a" chars will be replaced with "b" and all "b" chars
    will be replaced with "a"
    Examples:
    substitutions = ["ab"], input = "aabbcc", output = "bbaacc"
    substitutions = ["ab", "cd"], input = "adam", output = "bcbm"
